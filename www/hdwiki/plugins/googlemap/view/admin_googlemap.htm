欢迎使用Google Map 2.0插件！



本插件允许用户为词条添加相关标注点（可添加、可编辑）。 如果词条地图尚未添加标注点，则地图API默认会通过浏览器获取用户所在地理位置，并将地图中心移动到此地离位置。



安装说明：

  1. 按照 插件钩子 页中的说明在相应的页面添加钩子；
  2. 在hdwiki根目录下找到 /view/default/viewdoc.htm，在其中寻找 “<div id="block_right">”，在其上添加：“{$mapviewstr}”（不包含引号）；
  3. 现在，地图应该可以正常使用了；



二次开发指南：


注意：此项不建议新手操作，除非你真的知道你在做什么，否则会引起无法预知的错误。
插件程序中地图功能与显示相关代码位于/plugins/googlemap/model/googlemap.class.php 文件的 get_map_view_str() 函数中，可参考 Google Maps API v3 技术文档 进行二次开发。欢迎你提供功能更丰富的地图插件。
